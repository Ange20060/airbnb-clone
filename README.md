# Airbnb-clone
Clone de airbnb développé par Dev & Co Team.

## Stack
Frontend: HTML, TailwindCSS, JavaScript
Backend: PHP
Database: MySQL

## Objectif
Créer une plateforme adaptée au contexte africain pour la location de logements comme airbnb.
